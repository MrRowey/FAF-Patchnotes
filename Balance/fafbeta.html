FAF Beta

Welcome to the fafbeta Patchnotes. The changes listed in this patch note are subject to change.

The changes that are currently part of fafbeta are to evaluate the new balance in actual games.

We have been working hard to maintain this update until we feel the changes are ready for the main game in beta.

This page will be updated as often as possible to show what changes are live on fafbeta

-- The Balance Team

Land

Spirit

GitHub Pull Request: #4852

Brings the Spirit's speed in line with the other t1 scouts. This plays a small role in reducing effectiveness in tree breaking as well.

  • Max Speed: 5 → 4.6
  • Max Acceleration: 5 → 4.6

Scathis

GitHub Pull Request: #4975

This unit was far more inaccurate than desired or indicated by the reticle. Some shots still go outside the reticle, but with this change, this is fixed for the most part. Increasing it further buffed its performance against shielded bases too much.

  • Fixed Spread Radius: 80 → 70

Navy

Naval TMD

Github Issue:4984

Navy tmd get's a slight rework with the nerf of battleship/battlecruiser tmd and a buff of cruiser/carrier tmd. This will make it so bs's will need cruiser/carrier support to deal with larger quantities of missiles. Aeon tmd is unchanged considering it's uniqueness didn't allow for these adjustments.

Cruisers

  • Range: 50 → 60
  • RoF: 0.5 → 0.25

Carriers (Only Cybran)

  • Range: 50 → 60
  • RoF: 0.5 → 0.25

Battleships & Battlecruiser

  • Range: 50 → 31

Cybran Destroyer

We increased range for the Salem before and found it to still be lacking considering the nerf to their frigates. This should help bring their early navy in line with the other factions.

NEW Aeon Gun Range Upgrade

GitHub Pull Request: 5014

The Aeon gun range upgrade is quite powerful with the current cost. We have split it to allow it to still be strong later but not oppressive in the t1/early t2 phase of the game. This along with the OC nerf should make the Aeon gun com far less oppressive while still being viable.

First Range Upgrade

  • Energy Cost: 15000 → 6000
  • Mass Cost: 500 → 300
  • Build Time: 500 → 350
  • New Range: 35 → 30

Second Range Upgrade

  • Requires First Range upgrade
  • Energy Cost: 31000
  • Mass Cost: 900
  • Build Time: 1100
  • New Range: 35
